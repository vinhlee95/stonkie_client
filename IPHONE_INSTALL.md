## Troubleshooting:

**Don't see "Add to Home Screen"?**

- Check that you're on the actual website (not a search result)
- Try refreshing the page

**Icon not showing?**

- Check your home screen pages by swiping left
- Try searching for "Stonkie" using search

**App not loading?**

- Make sure you have an internet connection for first load
- After first load, it will work offline!
